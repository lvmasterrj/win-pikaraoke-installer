# win-pikaraoke-installer
An installer for the pikaraoke on Windows

To download, <a href="https://raw.githubusercontent.com/lvmasterrj/win-pikaraoke-installer/main/installer.bat">click here</a> with the right mouse button and chose "save link as...".

The file verifies and installs:
- ffmpeg
- Chrome
- Python
- Pikaraoke
